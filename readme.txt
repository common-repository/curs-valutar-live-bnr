== Description ==
Multumesc pentru ca ai ales pluginul: Curs Valutar Live BNR

Iti permite afisarea Cursului Valutar live al BNR pe orice pagina, articol, in widget sau sidebar.

Shortcode-ul [curs_bnr] permite afisarea cursului valutar oriunde doriti: in pagina, widget, sidebar etc.

== Installation ==
Urmeaza instructiunile pentru instalarea pluginului si afisarea cursului valutar:

* Instalati pluginul in wordpress
* Activati pluginul
* Oriunde doriti sa afisati cursul valutar BNR (in pagina, intr-un articol, in widget, in sidebar etc.) adaugati textul: [curs_bnr]

Shortcode-ul [curs_bnr] permite afisarea cursului valutar oriunde doriti in pagina.
Daca esti multumit de acest plugin te rog sa votezi 5 stele pe pagina pluginului: https://wordpress.org/plugins/curs-valutar-live-bnr/
